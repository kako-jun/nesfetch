# 使い方ガイド

## 起動

NESエミュレータまたはNES実機でROMファイルを読み込むと、最初のゲーム機ロゴが表示されます。

## 操作方法

| 操作 | 動作 |
|---|---|
| 十字キー上 | 前のロゴに切替（先頭→末尾にラップ） |
| 十字キー下 | 次のロゴに切替（末尾→先頭にラップ） |

A/Bボタンは使いません。上下キーだけで全ロゴを閲覧できます。

## 画面表示

- ロゴ：選択中のゲーム機のピクセルアート
- ページ番号：現在のロゴ番号（「NN」形式）
- ナビゲーション矢印：▲▼が常に表示

## 操作の流れ

1. 起動すると最初のロゴが表示される
2. 十字キー上下でロゴを切り替える
3. 16種類すべてを自由に行き来できる（ラップアラウンド）
