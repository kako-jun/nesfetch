# 使い方ガイド

## 起動

NESエミュレータまたはNES実機でROMファイルを読み込むと、ゲーム機の一覧メニューが表示されます。

## 操作方法

### メニュー画面

| 操作 | 動作 |
|---|---|
| 十字キー上 | カーソルを上に移動 |
| 十字キー下 | カーソルを下に移動 |
| Aボタン | 選択中のゲーム機のロゴを表示 |

メニューは29項目あり、画面に収まらない分は自動的にスクロールします。カーソルが画面の端に達すると、リストが上下にスクロールして残りの項目が現れます。

### ロゴ表示画面

| 操作 | 動作 |
|---|---|
| Bボタン | メニュー画面に戻る |

ロゴ表示中は、選択したゲーム機のピクセルアートが画面全体に表示されます。Bボタンを押すと、元のメニュー画面に戻ります。カーソル位置は保持されるので、続けて別のゲーム機を選べます。

## 操作の流れ

1. 起動するとメニュー画面が表示される
2. 十字キー上下で見たいゲーム機を選ぶ
3. Aボタンでロゴを表示する
4. Bボタンでメニューに戻る
5. 別のゲーム機を選んで繰り返し楽しむ
