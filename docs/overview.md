# nesfetch — NESで動くneofetch

## アプリの概要

**nesfetch** は、neofetch風のシステム情報表示をファミコン（NES）実機の上で再現するプログラムです。十字キーの上下だけで16種類のゲーム機のピクセルアートロゴを直接切り替えて閲覧できます。

## 解決したい課題

neofetchはターミナル文化の象徴的なツールで、「自分の環境をアスキーアートで見せる」楽しさがある。でも、それを見せる場所はいつも近代的なPCの画面の上。もし、世界で最も制約の厳しいプラットフォームのひとつであるファミコンで同じことができたら？

256x240ピクセル、同時発色数25色、RAM 2KBという極限の環境で「ゲーム機の情報を表示する」という、本来ありえない組み合わせを実現すること。それ自体がジョークであり、技術的な挑戦であり、レトロゲームへの愛情表現です。

## ビジョン

- **レトロハードでモダンな遊びをする**。neofetchという現代の文化をファミコンに持ち込む面白さ
- **主要ゲーム機を一覧できる**。ファミコンからPS5まで、16のゲーム機が一本のROMに収まる（将来的に29種まで拡張予定）
- **ピクセルアートとして美しい**。各ゲーム機のロゴがNESの制約の中で丁寧に描かれている
- **実機で動く**。エミュレータだけでなく、本物のファミコンカートリッジに焼いて動かせる
