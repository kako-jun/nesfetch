# ロードマップ

## 近い将来に届けたいこと

### ゲーム機のスペック情報表示

ロゴ表示画面に、そのゲーム機の基本スペック（発売年、解像度、色数など）をテキストで併記したいと考えています。neofetchが「OSのバージョンやCPU情報を見せる」ツールであるように、nesfetchも「そのゲーム機の情報を見せる」という本来の役割をより忠実に再現します。

### 収録ゲーム機の追加

現在の29種類に加えて、さらにゲーム機を追加したいと考えています。Lynx、Game Gear、Virtual Boy、Wii U、3DOなど、ゲームの歴史に足跡を残したハードウェアはまだまだあります。NESのROM容量との相談になりますが、できるだけ多くのゲーム機を収録したいところです。

### ロゴのアニメーション

ロゴ表示時に、簡単なアニメーション効果（フェードイン、スクロールインなど）を加えたいと考えています。NESの制約の中でどこまで表現できるかという技術的チャレンジでもあります。

## もう少し先の展開

### BGM・効果音の追加

メニュー操作時のカーソル移動音や、ロゴ表示時の短いジングルなど、音の演出を加えたいと考えています。NESの音源（矩形波2ch、三角波1ch、ノイズ1ch、DPCM 1ch）でどんな音が作れるか、それ自体が楽しい挑戦です。

### 他のレトロハード版

NES版が完成したら、同じコンセプトを他のレトロハードに展開するのも面白いかもしれません。ゲームボーイ版、スーパーファミコン版、メガドライブ版など、それぞれのハードの制約の中で同じ「ゲーム機一覧」を表現するという、プラットフォームごとの個性が見えてきます。
