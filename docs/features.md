# 主要機能

## 16種類のゲーム機を収録（PT0パターンテーブル）

現在、パターンテーブル0（PT0）に収まる16のゲーム機ロゴを収録しています。残り13種はパターンテーブル1（PT1）対応として create_chr.py にデータ準備済みですが、main.c 側の対応は未実装です。

**収録ゲーム機一覧（16種・PT0）**

| メーカー | ゲーム機 |
|---|---|
| 任天堂（据置） | NES、SNES、N64、GameCube、Wii、Wii U、Switch |
| 任天堂（携帯） | Game Boy、GBA、NDS、3DS |
| ソニー（据置） | PS1、PS2、PS3、PS4、PS5 |

**将来追加予定（13種・PT1）**

| メーカー | ゲーム機 |
|---|---|
| ソニー（携帯） | PS Vita |
| セガ | Genesis、Saturn、Dreamcast |
| マイクロソフト | Xbox、Xbox 360、Xbox One、Xbox Series X |
| その他 | Atari、PC Engine、Neo Geo、Steam Deck、WonderSwan |

## 直接切替ブラウジング

メニュー画面は廃止され、十字キーの上下だけでロゴを直接切り替えられます。A/Bボタンは不要で、起動直後からすぐにロゴを閲覧できます。先頭と末尾はラップアラウンド（循環）するため、どちらの方向にも途切れなく操作できます。

## ページ番号とナビゲーション矢印

画面には現在のページ番号「NN」が表示されます。上下のナビゲーション矢印（▲▼）は常に表示され、操作可能であることを示しています。

## フリッカー低減描画

ロゴ切替時、2回目以降の描画では変更のあった行だけをクリアすることで、画面全体のちらつきを抑えています。

## ピクセルアートロゴ表示

各ゲーム機のロゴがピクセルアートで表示されます。NESの厳しい色数・解像度の制約の中で、各ゲーム機の特徴を捉えたドット絵が描かれています。

## NESネイティブ動作

エミュレータ上での動作だけでなく、実際のNESハードウェアでの動作を前提に作られています。iNES Mapper 0（NROM）という最もシンプルな構成で、特殊なハードウェアを必要としません。カートリッジに焼けば、本物のファミコンで動きます。
