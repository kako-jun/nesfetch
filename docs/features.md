# 主要機能

## 29種類のゲーム機を収録

ファミコンからSteam Deckまで、ゲームの歴史を彩った29のハードウェアが一覧で並びます。任天堂・ソニー・セガ・マイクロソフト・その他メーカーまで、主要なゲーム機を網羅しています。

**収録ゲーム機一覧**

| メーカー | ゲーム機 |
|---|---|
| 任天堂（据置） | NES、SNES、N64、GameCube、Wii、Switch |
| 任天堂（携帯） | Game Boy、GBA、DS、3DS |
| ソニー（据置） | PS1、PS2、PS3、PS4、PS5 |
| ソニー（携帯） | PS Vita |
| セガ | Genesis、Saturn、Dreamcast |
| マイクロソフト | Xbox、Xbox 360、Xbox One、Xbox Series X |
| その他 | Atari、PC Engine、Neo Geo、Steam Deck、WonderSwan |

## スクロール可能なメニュー

29のゲーム機名がリスト表示され、十字キーで上下にスクロールできます。NESの画面に収まりきらない数のエントリを、自然な操作感で閲覧できます。現在選択中の項目はハイライトされ、どこにいるか一目でわかります。

## ピクセルアートロゴ表示

メニューからゲーム機を選ぶと、そのハードウェアのロゴがピクセルアートで表示されます。NESの厳しい色数・解像度の制約の中で、各ゲーム機の特徴を捉えたドット絵が描かれています。

## NESネイティブ動作

エミュレータ上での動作だけでなく、実際のNESハードウェアでの動作を前提に作られています。iNES Mapper 0（NROM）という最もシンプルな構成で、特殊なハードウェアを必要としません。カートリッジに焼けば、本物のファミコンで動きます。
