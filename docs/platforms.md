# 対応環境

## NES実機

本物のファミコン / NES ハードウェアで動作します。ROMをカートリッジに書き込めば、実機のコントローラーで操作できます。iNES Mapper 0（NROM）を使用しているため、特殊なマッパーチップは不要です。

## エミュレータ

| エミュレータ | サポート状況 |
|---|---|
| Mesen | 対応（推奨） |
| FCEUX | 対応 |
| Nestopia | 対応 |

上記以外のNESエミュレータでも、iNES Mapper 0に対応していれば動作します。

## ROMの仕様

| 項目 | 内容 |
|---|---|
| マッパー | iNES Mapper 0（NROM） |
| PRG-ROM | 32KB |
| CHR-ROM | 8KB |
| ミラーリング | 水平 |

最もシンプルなNESのROM構成を採用しています。追加のRAMや特殊チップを使わないため、互換性が高く、ほぼすべてのNES環境で動作します。

## 制約事項

- **解像度**: 256x240ピクセル（NESの標準解像度）
- **色数**: 同時発色数はNESのパレット制約に従います
- **入力**: NES標準コントローラーの十字キー、A・Bボタンのみ使用
- **セーブ機能なし**: 起動するたびにメニューの先頭から始まります
